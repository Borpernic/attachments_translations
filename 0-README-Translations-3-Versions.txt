Versions of Language Packs for Attachments-2.0
==============================================

INSTALL these files by unzipping them in the root directory of your Joomla!
installation.  All the language files will be put into their correct places.

NOTE: These language files are for Attachments-2.0+ only!  If you need a
      language file for Attachments 1.3.4, please contact the extension author:

EXTENSION AUTHOR: Jonathan Cameron - jmcameron@jmcameron.net

NOTE:  
  The 2.2 version of the language packs are good for all previous 2+ version.
