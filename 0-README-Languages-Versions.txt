Versions of Language Packs for Attachments-3 for Joomla 1.6/1.7/2.5+
====================================================================

INSTALL these files using the regular Joomla installer (no need to unzip manually).

COMPATIBLITY:

   * Language packs with versions 3.0.1 and 3.0.1.* are compatible with 
     Attachments 3.0, 3.0.1, 3.0.2, and 3.0.3 on Joomla 1.6, 1.7, and 2.5
   * Language packs with version 3.0.3 will work on Attachments 3.0.3 or 
     earlier versions.

NOTES: 

   * These language files are for Attachments-3.0+ only!  

   * If you need a language file for Attachments 2.2, please visit the other
     attachments project on joomlacode: http://joomlacode.org/gf/project/attachments/
